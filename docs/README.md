# PollNow Documentation

This directory contains the final documentation for the PollNow serverless polling application.

## Final deliverables

- [`final-project-summary.md`](final-project-summary.md) - overview of the final application, implemented features, AWS services and known limitations.
- [`final-architecture.md`](final-architecture.md) - detailed architecture reference covering frontend routes, API Gateway routes, Lambda functions, DynamoDB tables, S3 usage, SNS notifications, EventBridge and security model.
- [`final-test-checklist.md`](final-test-checklist.md) - end-to-end checklist for validating the deployed application before presentation or evaluation.

## Archive

Implementation-phase notes, historical checklists and detailed SES/SNS runbooks are preserved in [`archive/`](archive/). They are not required for a first review of the final project, but remain available for implementation traceability.
